Claude Code v2.1.88 ships 25+ fixes for long-session stability, adds flicker-free rendering

Today Claude Code Changelog

The biggest Claude Code release in weeks lands with fixes targeting the pain points of heavy daily users: prompt cache misses caused by tool schema bytes changing mid-session, nested CLAUDE.md files being re-injected dozens of times, a memory leak from large JSON inputs retained as LRU cache keys, and an OOM crash when editing files over 1 GiB. New features include CLAUDE_CODE_NO_FLICKER=1 for flicker-free alt-screen rendering, a PermissionDenied hook that fires after auto mode denials, and named subagents in @ mention typeahead.

The prompt cache miss fix is the headliner — cache misses in long sessions meant silently burning extra tokens and adding latency every time the tool schema drifted. This alone could meaningfully reduce costs for power users running multi-hour sessions.

Microsoft ships “Critique”: GPT drafts, Claude reviews — multi-model Copilot is here

Today GeekWire / Thurrott / PYMNTS

Microsoft launched three Copilot features today. “Critique” puts GPT and Claude in sequence: GPT generates a research response, then Claude reviews it for accuracy, completeness, and citation quality before the user sees it. “Model Council” lets users compare responses from different models side-by-side. And Copilot Cowork — built on Anthropic’s Claude Cowork technology — enters early access via the Frontier program, handling multi-step tasks across Outlook, Teams, and Excel.

GeekWire’s take: Microsoft is admitting no single model is good enough, but the 13.8% benchmark improvement over standalone Google and Perplexity tools suggests the multi-model approach isn’t just hedging — adversarial review between competing models genuinely produces better output.

Mistral borrows $830M from seven banks to build a 13,800-GPU data center outside Paris

Today Bloomberg / TechCrunch / CNBC

France’s Mistral secured $830 million in debt financing — backed by BNP Paribas, Crédit Agricole, HSBC, and four other banks — to fill a facility in Bruyères-le-Châtel with 13,800 Nvidia GB300 GPUs. The 44MW data center is expected to open in Q2 2026. Mistral aims for 200MW of capacity across Europe by end of 2027, following a €1.2 billion Swedish data center announcement in February.

Bloomberg’s reporting: this is the largest European AI infrastructure debt deal to date, and it’s pure debt, not equity — meaning Mistral is betting it can generate enough revenue from these GPUs to service $830M in loans, a fundamentally different risk profile than burning VC cash.

Shopify Agentic Storefronts go live — 5.6M merchants now shoppable inside ChatGPT, Gemini, and Copilot

Mar 24 Shopify / CNBC

Shopify’s “Agentic Storefronts” activated by default for all eligible US merchants on March 24. Products are now discoverable in ChatGPT, Google AI Mode, Microsoft Copilot, and the Gemini app — no separate integrations, no apps, no extra transaction fees. Buyers are redirected to the merchant’s own checkout. AI-driven traffic to Shopify stores has grown 7x since January 2025.

CNBC’s analysis: this is the first real agentic commerce infrastructure at scale — 5.6 million merchants became shoppable inside AI assistants overnight, and the “default on” design means most store owners didn’t have to do anything. The distribution advantage now belongs to whoever the AI recommends.
