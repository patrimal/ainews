Karpathy’s “autoresearch” — AI agent runs 700 experiments, catches bugs humans missed for 20 years

Today Andrej Karpathy / Fortune / VentureBeat

Karpathy open-sourced autoresearch, a 630-line Python script that gives an AI coding agent a small LLM training setup and lets it experiment autonomously. Over two days it ran ~700 experiments on nanochat, discovered 20 additive optimizations — including a missing scalar multiplier in QK-Norm attention — and cut the “Time to GPT-2” benchmark from 2.02 hours to 1.80 hours (11% faster). Shopify CEO Tobi Lütke tried it on internal data overnight: 37 experiments, 19% performance gain.

Karpathy’s own framing: “All LLM frontier labs will do this. It’s the final boss battle.” Fortune calls it “The Karpathy Loop” — the agent caught an attention scaling oversight that went unnoticed through two decades of manual optimization. This is AI doing useful AI research, not a benchmark game.

Claude Code v2.1.77: Opus 4.6 default output jumps to 64K tokens

Today Claude Code Changelog

Today’s Claude Code release raises the default max output for Opus 4.6 to 64K tokens, with the upper bound for both Opus 4.6 and Sonnet 4.6 now at 128K. Also adds an allowRead sandbox filesystem setting to re-allow read access within denyRead regions, and /copy now accepts an optional index to copy the Nth-latest assistant response.

64K default output means Opus can generate entire files, long multi-step analyses, and complex refactors in a single turn without hitting truncation — a practical quality-of-life upgrade for anyone using Claude Code for real work.

GTC keynote recap: Vera Rubin unveiled, Jensen projects $1 trillion in orders

Yesterday NVIDIA / CNBC / TechCrunch

Jensen Huang’s two-hour GTC keynote delivered the Vera Rubin platform: 10x performance per watt over Blackwell, connecting up to 144 GPUs via the new Kyber rack architecture (vertical compute trays for density and lower latency). Also announced: Groq 3 LPU partnership (256 LPUs per rack alongside Rubin), NemoClaw open-source agent platform, DLSS 5 for gaming, and a “Vera Rubin Space-1” plan for orbital data centers. Jensen’s headline number: $1 trillion in combined Blackwell and Vera Rubin purchase orders expected through 2027.

TechCrunch’s headline: Jensen “put projections into the $1 trillion stratosphere.” The Vera Rubin specs and Kyber architecture are genuinely novel engineering. The $1T number is Jensen being Jensen — it’s designed to move markets, not inform engineers.

NVIDIA + Uber bring robotaxis from BYD, Hyundai, and Nissan to the ride-hailing network

Yesterday NVIDIA GTC / Tom’s Guide

At GTC, NVIDIA announced four new robotaxi partners — BYD, Hyundai, Nissan, and Wayve — with Uber integrating autonomous vehicles into its app in select cities. The Nissan Leaf-based robotaxi service, developed with Wayve, will pilot on Uber in Tokyo in late 2026. All vehicles run on NVIDIA’s DRIVE platform.

This isn’t a research demo or a geofenced pilot — Uber connecting robotaxis from multiple manufacturers into its actual ride-hailing network means autonomous vehicles are entering mainstream consumer infrastructure, not just running closed test programs.
