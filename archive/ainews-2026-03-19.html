Hunter Alpha unmasked — it’s Xiaomi’s MiMo-V2-Pro, not DeepSeek V4

Today Japan Times / Technology.org / BusinessToday

The anonymous trillion-parameter model that appeared on OpenRouter last week — sparking a frenzy of DeepSeek V4 speculation — was revealed today as Xiaomi’s MiMo-V2-Pro. Luo Fuli, head of Xiaomi’s AI division MiMo and a former DeepSeek researcher, confirmed it’s an “early internal test build” designed to power autonomous AI agents. Xiaomi released three models simultaneously: MiMo-V2-Pro (language), MiMo-V2-Omni (multimodal), and MiMo-V2-TTS (speech). Xiaomi’s Hong Kong shares surged 5.8%.

The Japan Times notes this reshuffles assumptions about China’s AI landscape — a consumer electronics company is now training at trillion-parameter scale. The model topped OpenRouter’s leaderboard and processed over a trillion tokens during its anonymous run, proving Xiaomi’s AI ambitions go well beyond phone assistants.

Claude Code 2.1.79 — remote control bridge, session auto-naming, 18MB memory savings

Yesterday Claude Code Changelog

Yesterday’s Claude Code release adds /remote-control in VSCode to bridge your terminal session to claude.ai/code, so you can continue coding from a browser or phone. Sessions now get AI-generated tab names. Other additions: --console flag for Anthropic Console auth, “show turn duration” toggle, and fixes for claude -p hanging as a subprocess. Startup memory usage dropped ~18MB across all scenarios.

DevOps.com frames remote control as “keeping your agent local while putting it in your pocket” — the session still runs on your machine with full filesystem access, but any browser becomes a window into it. For developers who kick off long agent runs, this means you can leave your desk without losing context.

Anthropic v. Pentagon — Microsoft files amicus brief, hearing set for March 24

Mar 16 Axios / The Silicon Review / CNBC

Microsoft formally backed Anthropic’s lawsuit against the Pentagon’s supply-chain risk designation, filing an amicus brief urging a federal judge to grant a temporary restraining order. The filing argues the blacklisting — typically reserved for foreign adversaries like Huawei — threatens to disrupt critical AI capabilities across the defense ecosystem. Industry groups representing hundreds of companies have joined the push. A hearing on temporary relief is set for March 24.

Axios’s framing: this started as an Anthropic problem but has become an industry-wide concern. If the Pentagon can label a domestic AI company a “supply chain risk” for refusing to build autonomous weapons, every AI company’s relationship with the defense sector is up for renegotiation.
