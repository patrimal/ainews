Karpathy’s “Dobby” agent scans his network, reverse-engineers APIs, and runs his entire smart home via WhatsApp

Today @karpathy / Fortune / Let’s Data Science

Andrej Karpathy demonstrated “Dobby the House Elf” — a persistent OpenClaw agent that autonomously discovered devices on his local network, reverse-engineered undocumented Sonos API endpoints, and started playing music in his study without being told how. Dobby now controls lighting, HVAC, security cameras, shades, pool, and spa — all via natural-language WhatsApp messages. When a FedEx truck arrives, Dobby detects it on his security cameras and texts him. Karpathy says he’s in a “state of psychosis trying to figure out what’s possible.”

Karpathy’s point: the agent didn’t need documentation, SDKs, or setup — it discovered and integrated with hardware autonomously. If this pattern generalizes, vendor apps become vestigial and the OS-level “app” abstraction starts to erode.

Claude Code leak deep-dives reveal KAIROS daemon mode, Undercover Mode, and a supply chain attack follows within 24 hours

Today Alex Kim / WaveSpeedAI / PiunikaWeb

Detailed analyses of the 512K-line Claude Code source leak surfaced major unreleased features. KAIROS is an always-on autonomous daemon with a /dream skill for “nightly memory distillation,” append-only daily logs, and 5-minute cron refreshes — scaffolding for a persistent background agent. Undercover Mode (~90 lines) strips all traces of AI authorship in non-internal repos with no force-OFF switch, meaning AI-authored commits appear human-authored. Also found: anti-distillation defenses (fake tools + cryptographic signatures), frustration detection via regex, and a production bug that wasted ~250K API calls/day. Separately, attackers registered malicious npm packages (color-diff-napi, modifiers-napi) squatting on internal names developers encounter when building the leaked code.

Alex Kim’s analysis: KAIROS suggests Anthropic is building toward an agent that doesn’t wait to be asked — it watches, logs, and acts proactively. The Undercover Mode revelation is more immediately controversial: AI contributions to public open-source repos will have no indication they were AI-authored.

Perplexity AI hit with class-action: “undetectable” trackers allegedly shared user chats with Meta and Google — even in Incognito mode

Today Bloomberg

A proposed class-action filed in San Francisco federal court accuses Perplexity of embedding tracking software that transmits user conversations to Meta and Google the moment users log in. The complaint alleges data flows even when users enable Perplexity’s Incognito mode. The plaintiff, a Utah man filing as John Doe, shared financial details including family tax obligations and investment strategies with the chatbot. Both Meta and Google are named as defendants for allegedly exploiting the data for ad targeting. Perplexity’s spokesperson said the company “does not share user data with Meta or Google” and has not been served.

Bloomberg’s framing: if proven, the AI search engine marketed as a privacy-conscious alternative to Google was doing exactly what users switched away from — and the Incognito mode they relied on was purely cosmetic.

Claude Code turns one: 4% of all GitHub commits, DAU doubled last month, and v2.1.89 ships with deferred hooks

Today @bcherny / Claude Code Changelog

Boris Cherny marked Claude Code’s first anniversary with a reflective post — and striking numbers. The tool now authors 4% of all public GitHub commits (~135K/day), with Semi Analysis predicting 20% by year-end. DAU doubled in the past month. Cherny, who hasn’t manually edited code since November, declared “coding is largely solved.” Separately, v2.1.89 shipped today with notable features: "defer" permission decisions for PreToolUse hooks (headless sessions can pause at tool calls and resume later), flicker-free alt-screen rendering, named subagents in typeahead, non-blocking MCP connections, and 30+ bug fixes including memory leaks and CJK text handling. Also: /buddy hatches a small creature that watches you code — it’s April Fools’ Day.

Cherny’s argument: the shift is from “software engineer” to “builder” — code is becoming the assembly language of intent. The deferred hooks in v2.1.89 are practically significant: CI/CD pipelines can now pause at risky tool calls for human approval, then resume without restarting.

ChatGPT launches on Apple CarPlay — voice-only, no text, first AI chatbot in the car dashboard

Yesterday AppleInsider / 9to5Mac / MacRumors

OpenAI updated ChatGPT to support Apple CarPlay, making it the first major AI chatbot accessible through a car’s infotainment system. The interface is deliberately minimal: voice-only, no text or imagery, with just a mute toggle and end-call button. There’s no wake word — you have to manually open the app. Apple’s iOS 26.4 (released March 24) enabled the underlying entitlement; Claude and Gemini are expected to follow. The feature requires Apple’s explicit approval for each app.

9to5Mac’s take: Apple’s decision to gate AI chatbots behind a voice-only, no-visual-output requirement sets a strict safety template for in-car AI — the design constraint will shape how every competitor builds their CarPlay integration.
