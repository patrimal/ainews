Musk files to oust Altman and Brockman from OpenAI — trial starts April 27

Yesterday CNBC / Bloomberg / Gizmodo

Elon Musk’s lawyers filed amended remedies Tuesday seeking a court order to remove Sam Altman from OpenAI’s nonprofit board and both Altman and Greg Brockman as officers of the for-profit entity. In a tactical shift, Musk is no longer seeking the $134B in damages for himself — he wants any winnings returned to OpenAI’s original nonprofit. “Any assets obtained at the charity’s expense belong to the OpenAI charity and must be returned to it,” the filing states. OpenAI shot back that this is “pretending to change his tune” and filed its own request with California and Delaware attorneys general to investigate Musk for “anti-competitive behavior.” Jury selection begins April 27 in Oakland federal court; the trial is expected to run four weeks.

CNBC’s framing: Musk redirecting damages to the nonprofit neutralizes OpenAI’s best argument — that this is about money, not mission. Whether the jury buys it or not, the reframing forces OpenAI to defend the for-profit conversion on its merits.

Karpathy introduces “idea files” — says GitHub Gists beat X for real discourse

Apr 5 Andrej Karpathy / X / GitHub

After his LLM Wiki gist went viral, Karpathy formalized what he calls “idea files” — descriptions of concepts specific enough to be actionable but loose enough that an LLM agent can fill in the implementation details for your particular setup. The premise: in the era of LLM agents, you share the idea, not the code — the other person’s agent customizes and builds it. Then on April 5, he posted that he’s “surprised with how good the comments on GitHub gists are — a lot more helpful, insightful, constructive, a lot less AI” and asked GitHub to “consider competing with X.” The observation that lack of incentives (no likes, no followers, no algorithm) produces higher-quality discourse landed hard.

Karpathy’s point: the shareable unit in the LLM era is shifting from code to concept. An idea file is a prompt, a spec, and a tutorial rolled into one — and the platform where these get discussed matters as much as the ideas themselves.
